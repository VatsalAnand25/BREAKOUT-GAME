# 🎮 Breakout Game (Pygame)

A polished **Breakout-style arcade game** built with Python & Pygame, featuring modular architecture and clean game design.

---

## 🚀 Highlights

* 🧠 Object-Oriented Design (OOP)
* ⚡ Real-time physics & collision handling
* 🎯 Score + Lives system
* 🧱 Dynamic brick generation
* 🧩 Modular, scalable codebase

---

## 🎮 Controls

| Key | Action     |
| --- | ---------- |
| ←   | Move Left  |
| →   | Move Right |
| ESC | Exit       |
